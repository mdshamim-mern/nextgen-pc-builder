# NextGen PC Builder

A high-performance, AI-driven PC building platform designed to help users select compatible components, manage budgets, and build their dream PC with ease.

## 🚀 Features

- **AI-Powered Builder**: Intelligent compatibility checking for PC components.
- **Dynamic Filtering**: Advanced filtering system for CPUs, GPUs, RAM, and more.
- **Responsive Design**: Modern, clean UI built with Next.js and Tailwind CSS.
- **Performance Optimized**: Fast loading speeds with efficient component rendering.
- **User Authentication**: Secure user management for saving custom builds.

## 🛠 Tech Stack

### Frontend
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animation**: Framer Motion

### Backend
- **Framework**: Node.js & Express.js
- **Database**: MongoDB
- **Architecture**: RESTful API
